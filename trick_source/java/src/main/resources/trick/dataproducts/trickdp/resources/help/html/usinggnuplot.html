Using Gnuplot

By default the plots are displayed using a Fermi plot widget.

Gnuplot offers higher quality plots for reports.

To use Gnuplot, simply click the "Gnu" icon on the menu bar.

For quick postscript plots using Gnuplot click the icons that have red-lined buttons.

To switch the terminal type of Gnuplot: choose Settings->Gnuplot Terminal.

To edit the default titles on the gnuplot plots: choose Settings->Gnuplot->titles.
