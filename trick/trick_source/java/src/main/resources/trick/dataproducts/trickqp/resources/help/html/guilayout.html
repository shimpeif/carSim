GUI Layout

The "Vars" pane (top left) shows the variables available from the RUN to plot.
The "Runs" pane (lower left) shows the current RUN(s).
The "DP Tree" pane shows the DP spec file being built.
The "Property Notebook" on the bottom right shows the settings for plots, variables etc.
The parameters at the bottom allow data to be filtered by start/stop time and frequency (in seconds).
