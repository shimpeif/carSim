How to Organize Data

The trick_dp expects data to be organized in SIM_ prefixed directories.

Actual data should be in RUN_ directories.

Data Product spec files (DP_ files) must reside in a "DP_Product" directory.

A sample directoy tree is shown below:
trick_sims
   |
   |
   +---SIM_space_monkey
   |      +---- RUN_banana
   |      +---- RUN_tickle_self
   |      +---- DP_Product
   |               +- DP_tree.xml
   |               +- DP_bubble.xml
   |
   +---SIM_dog          
          +---- DP_Product
               +- DP_tree.xml
               +- DP_bubble.xml
          +---- RUN_test
